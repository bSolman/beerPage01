SpringBootLabs

Why
This project is mainly a learning project with the goal to learn the very basics of Spring Boot. That scope included features from the following dependencies.

	#spring-boot-starter-thymeleaf
	#spring-boot-starter-data-mongodb
	#spring-boot-starter-web
	#Lombok
	

Prerequisites
MongoDB collection. Preferrably located at a webserver somewhere. 
		
		
Installation
This project is not installed on any system, it is run locally inside the IDE.
